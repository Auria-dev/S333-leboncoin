## About this Project

This application is a functional recreation of the Leboncoin website, specifically focusing on the rental listings, tenant interactions, and landlord management modules.

This project was developed as part of a second-year university curriculum using the SCRUM agile methodology. The scope covered the complete software development lifecycle, including:

  * **Database Architecture:** Full modeling and implementation using PostgreSQL.
  * **UI/UX Design:** Interface design and user experience optimization.
  * **Compliance:** Implementation of RGPD (GDPR) data privacy standards.

## Technical Stack

The application is built using the **Laravel** PHP framework and utilizes **PostgreSQL** for database management.

It integrates various external services, including the **Google Maps API** for geolocation and listing visualization.
