## About this Project

This application is a functional recreation of the Leboncoin website, specifically focusing on the rental listings, tenant interactions, and landlord management modules.
Developed as part of a second-year university curriculum using the **SCRUM** agile methodology, the project covers the full software development lifecycle:

- **Database Architecture:** full modeling and implementation using PostgreSQL  
- **UI/UX Design:** interface design and user experience optimization  
- **Compliance:** RGPD (GDPR) data privacy standards  

## Technical Stack

- **Backend:** Laravel, PHP  
- **Database:** PostgreSQL  
- **Frontend:** Blade, JavaScript  
- **Mapping & Geolocation:** Leaflet
- **Interactive chatbot:** Botman & OpenAI
